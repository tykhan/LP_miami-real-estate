# Miami Real Estate - Landing Page

The landing page for Miami Kondo Kings real estate.

The project was created using:

- HTML5
- CSS3 (SCSS)
- Figma
- BEM-methodology

Landing page has the responsive layout for:

- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)
